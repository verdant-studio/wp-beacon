# wp-beacon

> Synchronize your sites metrics to a no-code databases.

> [!IMPORTANT]  
> This plugin is not yet ready for production.

## Requirements

- PHP >7.4
- Composer
- Node and NPM

## Getting started

```sh
# install node deps
npm i

# install composer deps
composer i
```

### Other commands

```sh
# watch css/js for admin
npm run watch

# build css/js for admin
npm run build

# unit test
composer run unit
```
